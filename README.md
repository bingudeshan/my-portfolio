# Deshan Hettiarachchi - Portfolio & Physics Blog

This shows a professional portfolio and physics blog built with React + Vite.

## Features
- **Sidebar Navigation**: Home, Physics Blog, Projects, and Experience.
- **Physics Blog**: Filterable list of posts with Markdown support.
- **Responsive Design**: Works on desktop and mobile.
- **Scientific Theme**: Dark mode with precise grid lines and glassmorphism effects.

## Setup Instructions

1.  Open a terminal in this folder.
2.  Install dependencies:
    ```bash
    npm install
    ```
3.  Run the development server:
    ```bash
    npm run dev
    ```
4.  Open your browser to `http://localhost:5173`.

## Customization
- Update your profile picture in `src/components/Sidebar.jsx`.
- Add new blog posts in `src/data/posts.json`.
- Edit `src/pages/Projects.jsx` to add more projects.

## Tech Stack
- React
- Vite
- Framer Motion (Animations)
- React Router (Navigation)
- React Markdown (Blog Content)
